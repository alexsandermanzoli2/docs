## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the lead to update
</ParamField>

<ParamField body="email" type="string">

<ParamField body="name" type="string">
  Lead's full name
</ParamField>

<ParamField body="phone" type="string">
  Lead's phone number with country code
</ParamField>

<ParamField body="company" type="object">
  Company information (partial updates supported)
  
  <Expandable title="Company Object">
    <ParamField body="name" type="string">
      Company name

    <ParamField body="domain" type="string">
      Company website domain
    </ParamField>
    
    <ParamField body="cnpj" type="string">
      Company CNPJ
    </ParamField>
    
    <ParamField body="industry" type="string">
      Industry classification
    </ParamField>
    
    <ParamField body="size" type="string">
      Company size
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="position" type="string">
  Lead's job title
</ParamField>

<ParamField body="linkedin_url" type="string">
  Lead's LinkedIn profile URL
</ParamField>

<ParamField body="status" type="string">

<ParamField body="remove_tags" type="array">

<ParamField body="custom_fields" type="object">
  Custom field values (merged with existing)

<ParamField body="notes" type="string">

## Update Operations

### Partial Updates

The PATCH method allows partial updates. Only include fields you want to change:

```json
{
  "score": 85
}
```

### Tag Management

You have three options for updating tags:

1. **Replace all tags** - Use the `tags` field
2. **Add tags** - Use the `add_tags` field
3. **Remove tags** - Use the `remove_tags` field

### Custom Fields

Custom fields are merged with existing values. To remove a custom field, set it to `null`:

```json
{
  "custom_fields": {
    "old_field": null,
    "new_field": "value"
  }
}
```

### Bulk Updates

To update multiple leads at once, use the bulk update endpoint:

```bash
POST /v1/leads/bulk-update
{
  "filter": {
    "tags": ["cold_lead"],
    "status": "active"
  },
  "update": {
    "status": "unqualified",
    "add_tags": ["archived"]
  }
}
```
