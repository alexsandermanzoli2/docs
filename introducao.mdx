# Bem-vindo à API Speedio

A API Speedio oferece acesso programático à nossa plataforma de geração e enriquecimento de leads B2B. Com nossa API, você pode buscar informações detalhadas sobre empresas brasileiras, validar dados e enriquecer suas bases de leads.

## URL Base da API

```
https://api-get-leads.speedio.com.br
```

## Autenticação

A API utiliza **Basic Authentication**. Você precisará de credenciais de acesso fornecidas pela Speedio.

```bash
Authorization: Basic base64(username:password)
```

<Warning>
  **Importante**: Por motivos de segurança, sempre use variáveis de ambiente para armazenar suas credenciais. Nunca inclua credenciais diretamente no código.
</Warning>

## Formato das Respostas

Todas as respostas da API são retornadas em formato JSON. 

### Exemplo de Resposta Bem-sucedida

```json
[
  {
    "cnpj": "21071712000171",
    "razao_social": "SPEED IO - SERVICOS ESPECIALIZADOS LTDA",
    "nome_fantasia": "SPEEDIO",
    "location": {
      "uf": "ES",
      "city": "Colatina",
      "bairro": "CENTRO",
      "cep": "29700010"
    },
    "website": "https://speedio.com.br/",
    "qnt_funcionario_empresa": "51 a 100",
    "faixa_faturamento_empresa": "10M a 30M"

```

## Rate Limits

- **100 requisições por minuto** por conta
- **1000 requisições por hora** por conta
- Headers de resposta incluem informações sobre limites restantes
