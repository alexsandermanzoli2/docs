## Visão Geral

A API de Parceiros da Speedio oferece acesso privilegiado a recursos avançados de busca e enriquecimento de dados. Esta API é exclusiva para parceiros autorizados e fornece informações detalhadas sobre empresas e seus contatos.

## URL Base

Todas as requisições para a API de Parceiros devem ser feitas para:

```
https://api-get-leads.speedio.com.br
```

## Autenticação

A API de Parceiros utiliza autenticação Basic Auth. Você receberá credenciais específicas ao se tornar um parceiro autorizado.

```bash
Authorization: Basic base64(username:password)
```

### Limites por Plano

| Plano | Requisições/Minuto | Requisições/Dia | CNPJs por Requisição |

| Starter | 60 | 10.000 | 10 |

| Custom | Personalizado | Personalizado | Personalizado |

## Diferenças da API Principal

A API de Parceiros oferece:

- **Dados mais detalhados**: Informações completas sobre empresas e contatos
- **Busca por CNPJ**: Capacidade de buscar diretamente por CNPJ
- **Enriquecimento avançado**: Dados adicionais não disponíveis na API principal
- **SLA diferenciado**: Tempo de resposta garantido para parceiros

## Como se Tornar um Parceiro

Para ter acesso à API de Parceiros:

1. Entre em contato com nossa equipe comercial
2. Apresente seu caso de uso e volume esperado
3. Assine o acordo de parceria
4. Receba suas credenciais exclusivas
