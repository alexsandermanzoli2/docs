## Visão Geral

Este endpoint permite buscar leads qualificados em nossa base de dados usando filtros avançados como localização, cargo, empresa, setor e muito mais.

## Requisição

### URL
```
GET https://api-get-leads.speedio.com.br/search_leads
```

### Parâmetros de Query

<ParamField query="location" type="string">
  Localização geográfica (cidade, estado ou região)
  
  Exemplos: `"São Paulo, SP"`, `"Rio de Janeiro"`, `"Sul"`
</ParamField>

<ParamField query="job_title" type="string">
  Cargo ou função do lead
  
  Exemplos: `"CEO"`, `"Diretor"`, `"Gerente de Vendas"`
</ParamField>

<ParamField query="company_name" type="string">
  Nome da empresa
  
  Exemplo: `"Microsoft"`, `"Magazine Luiza"`
</ParamField>

<ParamField query="industry" type="string">
  Setor da empresa
  
  Exemplos: `"Tecnologia"`, `"E-commerce"`, `"Saúde"`
</ParamField>

<ParamField query="company_size" type="string">
  Tamanho da empresa por número de funcionários
  
  Opções: `"1-10"`, `"11-50"`, `"51-200"`, `"201-1000"`, `"1000+"`
</ParamField>

<ParamField query="seniority" type="string">
  Nível de senioridade
  
  Opções: `"c-level"`, `"vp"`, `"director"`, `"manager"`, `"specialist"`
</ParamField>

<ParamField query="department" type="string">
  Departamento
  
  Exemplos: `"sales"`, `"marketing"`, `"technology"`, `"finance"`
</ParamField>

<ParamField query="email_verified" type="boolean">
  Filtrar apenas leads com e-mail verificado
  
  Padrão: `true`
</ParamField>

<ParamField query="phone_verified" type="boolean">
  Filtrar apenas leads com telefone verificado
  
  Padrão: `false`
</ParamField>

<ParamField query="limit" type="integer">
  Número máximo de resultados por página
  
  Mínimo: `1`, Máximo: `100`, Padrão: `20`
</ParamField>

<ParamField query="offset" type="integer">
  Número de registros para pular (paginação)
  
  Padrão: `0`
</ParamField>

<ParamField query="sort" type="string">
  Campo para ordenação
  
  Opções: `"created_at"`, `"updated_at"`, `"score"`, `"company_size"`
  
  Prefixar com `-` para ordem decrescente
</ParamField>

```python Python
import requests
import base64
import os

# Configure as variáveis de ambiente
username = os.getenv('SPEEDIO_USERNAME')
password = os.getenv('SPEEDIO_PASSWORD')

if not username or not password:
    raise ValueError('SPEEDIO_USERNAME e SPEEDIO_PASSWORD devem estar definidas')

# Configurar autenticação
auth = base64.b64encode(f"{username}:{password}".encode()).decode()

# Parâmetros de busca
params = {
    'location': 'São Paulo, SP',
    'job_title': 'CEO',
    'company_size': '51-200',
    'industry': 'Tecnologia',
    'email_verified': True,
    'limit': 10,
    'sort': '-score'
}

# Headers
headers = {
    'Content-Type': 'application/json',
    'Authorization': f'Basic {auth}'
}

# Fazer requisição
response = requests.get(
    'https://api-get-leads.speedio.com.br/search_leads',
    params=params,
    headers=headers
)

if response.status_code == 200:
    data = response.json()
    print(f"Encontrados {data['pagination']['total']} leads")
    
    for lead in data['data']:
        print(f"{lead['name']} - {lead['job_title']} - {lead['company']['name']}")
else:
    print(f"Erro: {response.status_code}")
    print(response.text)
```

## Filtros Avançados

### Combinação de Filtros

Você pode combinar múltiplos filtros para refinar sua busca:

```bash
# Buscar CTOs de empresas de tecnologia em SP com mais de 100 funcionários
curl 'https://api-get-leads.speedio.com.br/search_leads?location=São Paulo&job_title=CTO&industry=Tecnologia&company_size=201-1000'
```

### Operadores de Busca

Use operadores especiais para buscas mais precisas:

| Operador | Descrição | Exemplo |
|----------|-----------|---------|
| `*` | Wildcard | `job_title=*Diretor*` |
| `!` | Negação | `industry=!Varejo` |
| `\|` | OU lógico | `job_title=CEO\|CTO\|CFO` |

### Filtros por Score

```bash
# Buscar apenas leads com score alto
curl 'https://api-get-leads.speedio.com.br/search_leads?score_min=80&email_verified=true'
```

## Paginação

Para navegar pelos resultados:

```bash
# Primeira página (0-19)
curl 'https://api-get-leads.speedio.com.br/search_leads?limit=20&offset=0'

# Segunda página (20-39)
curl 'https://api-get-leads.speedio.com.br/search_leads?limit=20&offset=20'

# Terceira página (40-59)
curl 'https://api-get-leads.speedio.com.br/search_leads?limit=20&offset=40'
```

## Ordenação

Ordene os resultados por diferentes campos:

```bash
# Por score (maior primeiro)
curl 'https://api-get-leads.speedio.com.br/search_leads?sort=-score'

# Por data de criação (mais recente primeiro)
curl 'https://api-get-leads.speedio.com.br/search_leads?sort=-created_at'

# Por tamanho da empresa (maior primeiro)
curl 'https://api-get-leads.speedio.com.br/search_leads?sort=-company_size'
```

## Limites e Custos

- **Máximo de resultados por requisição**: 100
- **Custo por lead retornado**: 0.5 créditos
- **Rate limit**: Conforme seu plano de parceiro

<Info>
  Você só é cobrado pelos leads únicos retornados, não por requisições duplicadas do mesmo lead.
</Info>
